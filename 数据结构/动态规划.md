## 斐波那契数列
### 递归公式
```java
  int fib(int n) {
    return n <= 1 ? n : fib(n - 1) + fib(n - 2);
  }
```

### 时间复杂度
O(2^n)

### 记忆化
加一个缓存，O(n)

### 递推公式
```java 
  f[0] = 0;
  f[1] = 1;
  for (int i = 2; i <=n; ++i) {
    f[i] = f[i - 1] + f[i - 2]
  }
```

## 对比
DP VS 回溯 VS 贪心

- 回溯（递归） - 重复计算

- 贪心 - 永远局部最优

- DP - 记录局部最优子结构/多种记录值
