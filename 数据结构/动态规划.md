## “一个模型三个特征”理论讲解
**一个模型**指的是动态规划适合解决的问题的模型

**三个特征**

- 最优子结构

问题的最优解包含子问题的最优解，也可以通过子问题的最优解推导出问题的最优解

- 无后效性

推导后面状态的时候，我们只关心前面阶段的状态值，不关心这个状态是怎么推导出来的

- 重复子问题

===>

- 1.递归 + 记忆化 -> 递推

- 2.状态的定义：opt[n]，dp[n]，fib[n]

- 3.状态转移方程： opt[n] = best_of(opt[n - 1], opt[n - 2], ...)

- 4.最优子结构


## 斐波那契数列
### 递归公式
```java
  int fib(int n) {
    return n <= 1 ? n : fib(n - 1) + fib(n - 2);
  }
```

### 时间复杂度
O(2^n)

### 记忆化
加一个缓存，O(n)

### 递推公式
```java 
  f[0] = 0;
  f[1] = 1;
  for (int i = 2; i <=n; ++i) {
    f[i] = f[i - 1] + f[i - 2]
  }
```

## 对比
DP VS 回溯 VS 贪心

- 回溯（递归） - 重复计算

- 贪心 - 永远局部最优

- DP - 记录局部最优子结构/多种记录值
