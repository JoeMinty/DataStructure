## 异或（XOR）
x ^ 0 = x

x ^ 1s = ~x

x ^ (~x) = 1s

x ^ x = 0

a ^ b = c => a ^ c = b, b ^ c = a // 交换


## 实战
- X & 1 == 1 OR X & 1 == 0 判断奇偶（X % 2 == 1）

- X = X & (X - 1) => 清零最低位的1

- X & -X => 得到最低位的1

## 复杂的位运算操作
- 将x最右边的n位清零

**x & (~0 << n)**

- 获取x的第n位值（0或者1）

**(x >> n) & 1**

- 获取x的第n位的幂值

**x & (1 << (n - 1))**

- 仅将第n位置为1

**x | (1 << n)**

- 仅将第n位置位0

**x & (~(1 << n))**

- 将x最高位值第n位（包含该位）清零

**x & ((1 << n) - 1)**

- 将第n位至第0位（含）清零

**x & (~((1 << (n + 1)) -1))**
